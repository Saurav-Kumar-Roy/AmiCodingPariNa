Backend (Django rest framework):
Create a virtual envionment with python.
make the virtual environment active.
goto backend folder of the project
open in terminal
give comand "pip install -r requriment.txt"
after all pakage are installed use "python manage.py runserver" to start the backend.

Frontend (react):
go to frontend folder
open in terminal and give command "npm i"
after installing all the pakage use "npm start" to start the server. by defalut it will open at localhost:3000. go to the address and the app ui will be visible.
